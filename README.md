# Will you be my Valentine?
#### Video Demo:  https://youtu.be/FpYCSQR1j8A
#### Description: 
This program will ask the user if they want to be my valentine. If the user types 'yes', a cute webpage will be opened. If the user types 'no', a different webpage will be opened to ask the user to make a choice again. If the user types any other answers, the program will remind the user that only 'yes' and 'no' (case-insensitive) can be entered.
